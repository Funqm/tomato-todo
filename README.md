# tomato-todo

## 功能和样式参照[专注清单](https://www.focustodo.cn/?lang=zh_CN)

- [x] 任务添加，设置番茄时间，任务，标签
- [x] 执行任务，记录专注时间
- [x] 定时器动画
- [x] 标签选择，区分用户自定义标签和原生标签
- [ ] 添加自定义标签功能
- [ ] 任务详情功能
- [ ] 用户设置页面
